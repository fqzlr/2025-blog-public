旧笔记本安装飞牛NAS（fnOS）超详细教程【Ventoy版】

前言：家里闲置旧笔记本别再吃灰啦！用Ventoy制作启动盘，一步到位安装飞牛NAS（fnOS），零成本搭建私有云，存照片、存影视、做文件备份全搞定。全程图形化操作，小白也能轻松上手，每一步都标清重点，跟着做不踩坑 ✅

一、安装前准备（必做，缺一不可）

1. 硬件准备（旧笔记本为主，外设辅助）

- 核心设备：旧笔记本（x86架构，Intel/AMD处理器均可），要求内存≥4GB（推荐8GB+，多开功能不卡顿），硬盘≥64GB（优先SSD，启动/读写更快；机械硬盘也可，适合纯存储）。

- 关键外设：8GB及以上U盘（用于制作Ventoy启动盘，U盘数据会被清空，提前备份重要文件）。

- 必备配件：网线（飞牛NAS安装仅支持有线网络，无线暂不支持，避免安装失败）、显示器+键盘（鼠标可选，安装时键盘操作即可）。

- 可选扩展：移动硬盘/2.5寸硬盘盒（用于扩展存储空间，存大量影视、相册首选）、散热底座（旧笔记本长时间运行易发热，优化散热更耐用）。

2. 软件准备（提前下载，避免安装时断网）

- 飞牛NAS镜像：官网下载最新x86版ISO文件（官网地址：https://www.fnnas.com/download，选择“x86_64 镜像”，下载后无需解压）。

- Ventoy工具：免费开源的启动盘制作工具（官网地址：https://www.ventoy.net/cn/download.html，下载对应Windows版本，解压即可使用，无需安装）。

3. 笔记本BIOS关键设置（重中之重，避免安装失败）

不同品牌笔记本进入BIOS的快捷键不同，开机时屏幕会提示（常见：联想F2、戴尔F2、惠普F10、华硕F2/F8、宏碁F2），按对应按键进入，修改以下4个设置：

1. 关闭Secure Boot（安全启动）：找到“Security”（安全）选项，找到Secure Boot，设置为“Disabled”（禁用），否则无法识别飞牛镜像。

2. 硬盘模式改为AHCI：找到“SATA Controller Mode”（SATA控制器模式），选择“AHCI”，默认可能是RAID模式，会导致安装时识别不到硬盘。

3. 关闭Fast Boot（快速启动）：找到“Boot”（启动）选项，关闭Fast Boot，避免U盘启动异常，无法进入安装界面。

4. 保存设置并退出：按F10（部分机型需配合Fn），选择“YES”保存，笔记本会自动重启，重启后BIOS设置生效。

二、用Ventoy制作飞牛NAS启动盘（最简单，一次制作可多用）

Ventoy的优势：无需反复格式化U盘，只需将ISO镜像复制到U盘，即可启动安装，后续装其他系统也能复用，适合经常折腾的朋友。

1. 插入准备好的U盘，打开解压后的Ventoy工具（以Windows版本为例，双击“Ventoy2Disk.exe”运行）。

2. 工具会自动识别插入的U盘（注意：确认是自己的U盘，避免选错格式化错硬盘），核对“设备”选项是否正确。

3. 无需修改其他设置（默认分区类型GPT，适配新老笔记本；老BIOS笔记本可手动改为MBR），直接点击“安装”。

4. 弹出提示“该操作将格式化U盘，所有数据将丢失”，确认U盘已备份后，点击“确定”，等待2-3分钟，Ventoy安装完成（提示“安装成功”即可）。

5. 打开电脑文件管理器，找到刚制作好的“Ventoy”U盘（盘符会显示为Ventoy），直接将下载好的飞牛NAS ISO镜像文件，复制到U盘根目录（无需解压、无需改名，复制完成即可）。

6. 启动盘制作完成，拔掉U盘备用（后续插入旧笔记本使用）。

三、旧笔记本安装飞牛NAS系统（核心步骤，图文逻辑，跟着走）

步骤1：连接设备，准备启动

先将旧笔记本关机，依次连接好以下设备，避免中途拔插导致失败：

- 接好笔记本电源（避免安装中途断电，导致系统损坏）；

- 连接显示器、键盘（旧笔记本屏幕可正常使用的话，也可不用外接显示器，键盘必接）；

- 插入网线（一端接笔记本网口，一端接路由器LAN口，确保局域网通畅）；

- 插入制作好的Ventoy启动盘（U盘）。

步骤2：从U盘启动，进入Ventoy菜单

1. 开机，连续按笔记本对应的“启动项快捷键”（和进入BIOS的快捷键不同，常见：F12、F8、Esc），调出启动项选择菜单（Boot Menu）。

2. 用方向键选择你的U盘（通常显示“USB”“Ventoy”或U盘品牌，比如“Kingston DataTraveler”），按回车键确认，从U盘启动。

3. 启动后自动进入Ventoy菜单，菜单中会显示U盘根目录下的飞牛NAS ISO镜像（比如“fnOS-x86_64-xxx.iso”），用方向键选中该镜像，按回车键进入飞牛安装界面。

步骤3：图形化安装飞牛系统（关键步骤，重点标注）

1. 进入飞牛安装菜单后，选择“Graphical Install（图形化安装）”，按回车键（默认选中，直接回车即可），等待系统加载（约1-2分钟，耐心等待）。

2. 语言选择：默认“中文（简体）”，无需修改，按回车键继续。

3. 地区/时区选择：默认“中国”“上海”，按回车键继续（无需手动修改，避免时区错乱）。

4. 键盘布局：默认“汉语”，按回车键继续（适配普通键盘，无需调整）。

5. 选择安装磁盘（重中之重，别选错！）：
        

  - 进入“选择安装磁盘”界面，用方向键选中旧笔记本的内置硬盘（通常显示为“/dev/sda”，后面会标注硬盘容量，比如“128GB SSD”，区分开U盘和外接硬盘）。

  - 选中后按回车键，会弹出提示“此磁盘将被格式化，所有数据将丢失”，确认内置硬盘无重要数据（已提前备份）后，按回车键继续。

6. 分区设置：默认“64GB系统分区+自动Swap分区”（Swap分区用于内存扩展，无需手动修改），直接按回车键确认，系统开始自动分区、格式化并安装（约5-10分钟，取决于硬盘速度，耐心等待，别关机、别拔设备）。

步骤4：网络配置（必须记录IP地址，后续要用）

1. 系统安装完成后，会自动进入网络配置界面，默认识别已连接的有线网卡（显示为“eth0”或“enp0s3”），无需手动选择。

2. 选择“自动获取IP（DHCP）”，按回车键保存（路由器开启DHCP即可自动获取，无需手动设置IP）。

3. 配置完成后，屏幕会显示NAS的IP地址（比如“192.168.1.105”），务必用笔或手机记下来！这是后续登录管理界面的关键地址，丢了会很麻烦。

步骤5：完成安装，重启NAS

1. 网络配置完成后，会提示“请移除安装U盘”，拔掉插入的Ventoy启动盘（避免重启后再次进入安装界面）。

2. 选择“重启系统”，按回车键，笔记本会自动重启（约1-2分钟）。

3. 重启完成后，笔记本屏幕会显示飞牛NAS的待机界面，界面上会再次显示管理地址（就是刚才记录的IP地址），此时NAS系统已安装成功，笔记本已经变成一台NAS设备了。

四、飞牛NAS初始化配置（浏览器操作，简单易懂）

NAS系统安装完成后，后续配置都在浏览器中操作，无需再用显示器和键盘（配置完成后，显示器、键盘可拔掉），用同一局域网的电脑/手机即可操作。

步骤1：访问NAS管理页面

1. 打开同一局域网内的电脑/手机（和NAS连同一个路由器），打开任意浏览器（Chrome、Edge、微信浏览器均可）。

2. 在浏览器地址栏中，输入刚才记录的NAS IP地址（比如“http://192.168.1.105”），按回车键，进入飞牛NAS的初始化界面。

步骤2：设置管理员账号（牢记密码）

1. 初始化界面第一个步骤，创建管理员账号：输入“管理员用户名”（推荐用admin，好记）。

2. 设置管理员密码（要求：字母+数字+符号，比如“Admin@123456”，牢记密码，后续登录管理界面需要）。

3. 勾选“我已阅读并同意用户协议”，点击“下一步”。

步骤3：存储池与共享文件夹设置（核心功能，必做）

存储池就是NAS的“存储空间”，共享文件夹用于电脑/手机访问NAS里的文件，设置步骤如下：

1. 进入“存储管理”界面，系统会自动识别NAS的内置硬盘（如果外接了移动硬盘，也会显示），选中要使用的硬盘（推荐先用水内置硬盘）。

2. 点击“创建存储池”，默认设置即可（无需手动分区、无需调整参数），点击“确认”，等待存储池创建完成（约1-2分钟）。

3. 创建共享文件夹：点击“共享文件夹”→“新建”，输入文件夹名称（比如“文件备份”“家庭相册”“影视库”），设置访问权限（默认“管理员可读写，其他用户只读”，可根据需求调整），点击“确认”。

4. 开启文件访问协议：在“共享设置”中，开启SMB协议（用于Windows电脑、手机访问）和NFS协议（可选，用于Linux电脑访问），开启后保存设置。

步骤4：开启基础功能，完成初始化

1. 开启远程访问：进入“网络设置”→“FN Connect”，点击“一键开启”，开启后，外网也能访问NAS（比如出门在外，用手机访问家里的NAS文件）。

2. 开启常用功能：根据需求，开启“相册自动备份”（手机照片可自动备份到NAS）、“影视刮削”（自动识别影视信息，生成海报、剧情简介）、“Docker”（可搭建更多服务，比如导航页、下载器，小白可后续再折腾）。

3. 点击“完成初始化”，进入飞牛NAS主界面，此时，你的旧笔记本NAS已经完全可用了！

五、旧笔记本NAS优化（提升体验，延长使用寿命）

旧笔记本长时间运行NAS，做好以下优化，体验更好、更耐用，针对旧笔记本的痛点专门调整：

- 关闭屏幕，避免耗电：配置完成后，可合上笔记本盖子，进入BIOS设置“合上盖子不睡眠”（找到“Power”选项，设置“Lid Close Action”为“Do Nothing”），既省电又能避免屏幕长时间亮着损耗。

- 散热优化：旧笔记本散热普遍一般，长时间运行会发热卡顿，可将笔记本垫高（用支架或书本），或外接散热底座，避免硬盘、CPU高温损坏。

- 设置固定IP：进入路由器管理界面，找到NAS的设备（通过MAC地址识别，NAS待机界面可查看MAC），将NAS的IP地址绑定（静态IP），避免后续IP变动，无法访问NAS。

- 外接存储扩展：如果内置硬盘容量不够，用USB3.0接口接移动硬盘/硬盘盒（USB3.0速度更快），在NAS管理界面中添加为“扩展存储”，扩容更方便。

- 关闭无用服务：小白用户可关闭Docker、FTP等暂时用不到的服务，减少内存占用，让NAS运行更流畅。

六、常见问题解决（避坑指南，安装失败必看）

整理了安装过程中最常遇到的4个问题，对应解决方法，不用到处查资料：

- 问题1：安装时识别不到硬盘：大概率是BIOS中SATA模式没改为AHCI，重新进入BIOS，修改为AHCI模式，关闭RAID模式，重启后重新安装即可。

- 问题2：无法从U盘启动，进不去Ventoy菜单：检查3点：① U盘Ventoy制作是否成功（重新打开Ventoy工具，查看是否“安装成功”）；② BIOS中是否关闭Secure Boot和Fast Boot；③ 换一个USB口（优先插笔记本后置USB口，供电更稳定）。

- 问题3：网络获取不到IP，屏幕不显示IP地址：检查网线是否插好（两端都插紧），路由器是否开启DHCP功能，换一根网线或路由器LAN口，重新配置网络即可。

- 问题4：系统安装失败，提示“安装中断”：拔掉U盘，重启笔记本，重新进入Ventoy菜单，选中飞牛镜像，按Tab键，选择“Rescuing Install（应急安装）”，重试安装即可；若还是失败，重新下载飞牛ISO镜像（可能镜像损坏），重新复制到U盘。
