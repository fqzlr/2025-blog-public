## 1. 准备工作

### 🔧 硬件需求

| 物品 | 要求 | 说明 |
|------|------|------|
| 旧笔记本 | x86架构 | Intel/AMD CPU均可 |
| U盘 | 8GB以上 | 用于制作启动盘 |
| 网线 | 1根 | 推荐使用有线连接 |
| 显示器/键盘 | 临时使用 | 安装过程中需要 |
| 硬盘 | SSD+HDD | 系统盘建议SSD，数据盘用HDD |

### 💾 软件准备

1. **飞牛fnOS镜像**
   - 官网下载：[https://www.fnnas.com](https://www.fnnas.com)
   - 选择对应版本的ISO镜像文件

2. **Ventoy工具**
   - 官网下载：[https://www.ventoy.net](https://www.ventoy.net)
   - 支持Windows/Linux/Mac

3. **其他工具（可选）**
   - MobaXterm（远程管理）
   - Rufus（备用写盘工具）

## 2. 制作Ventoy启动U盘

### 步骤一：安装Ventoy到U盘

```
1. 下载并解压 Ventoy 工具包
2. 插入U盘（⚠️ 会清空所有数据，请提前备份）
3. 运行 Ventoy2Disk.exe（Windows）
4. 选择目标U盘
5. 点击「安装」按钮
6. 等待安装完成
```

### 步骤二：复制fnOS镜像

```
1. 打开U盘（安装Ventoy后会有两个分区）
2. 将下载的 fnOS-xxx.iso 文件直接复制到U盘
3. 安全弹出U盘
```

> ✅ **Ventoy优势**：无需反复格式化，支持多ISO文件共存，直接拖拽即可

## 3. BIOS/UEFI设置

### 进入BIOS方法

| 品牌 | 启动菜单键 | BIOS设置键 |
|------|-----------|-----------|
| 联想 | F12 | F2/Fn+F2 |
| 戴尔 | F12 | F2 |
| 惠普 | F9 | F10 |
| 华硕 | F8 | F2/Del |
| 宏碁 | F12 | F2 |

### 关键设置项

```
1. 关闭 Secure Boot（安全启动）
2. 开启 UEFI 启动（或 Legacy/CSM 兼容模式）
3. 设置U盘为第一启动项
4. 保存并退出（通常按F10）
```

## 4. 安装fnOS系统

### 启动安装程序

```
1. 插入制作好的Ventoy U盘
2. 开机按启动菜单键（如F12）
3. 选择U盘启动
4. 在Ventoy菜单中选择 fnOS 镜像
5. 按回车开始加载
```

### 图形化安装流程

```
┌─────────────────────────────────────────┐
│  1. 选择安装语言 → 中文                 │
│  2. 选择安装磁盘 → 选择目标硬盘          │
│  3. 确认分区方案 → 默认即可             │
│  4. 设置管理员密码 → 牢记！             │
│  5. 等待安装完成 → 约5-10分钟           │
│  6. 重启系统 → 拔出U盘                  │
└─────────────────────────────────────────┘
```

> ⚠️ **注意**：安装过程会清空目标硬盘所有数据！

```bash

# 进入root权限
sudo -i

# 修改合盖设置（不同系统可能不同）
# 编辑 systemd 配置
nano /etc/systemd/logind.conf

# 修改以下参数：
HandleLidSwitch=ignore
HandleLidSwitchExternalPower=ignore
HandleLidSwitchDocked=ignore
```

## 7. 常见问题解决

### ❌ 问题1：无法从U盘启动

**解决方案：**
```
1. 检查BIOS中是否关闭Secure Boot
2. 尝试切换UEFI/Legacy启动模式
3. 重新制作Ventoy U盘
4. 更换U盘或USB接口
```

### ❌ 问题2：安装时找不到硬盘

**解决方案：**
```
1. 检查硬盘连接是否正常
2. 部分笔记本需要加载额外驱动
3. 尝试更换SATA模式（AHCI/IDE/RAID）
4. 在BIOS中开启硬盘控制器
```

### ❌ 问题3：无法获取IP地址

**解决方案：**
```
1. 检查网线是否插好
2. 路由器DHCP是否开启
3. 尝试静态IP配置
4. 更换网口测试
```

## ⚠️ 注意事项

1. **数据无价**：安装前务必备份重要数据
2. **硬盘选择**：系统盘用SSD，数据盘用HDD（HDD损坏后数据更易恢复）
3. **网络稳定**：建议使用有线网络连接
4. **定期更新**：关注系统更新和安全补丁
5. **电源保护**：建议配置UPS防止断电损坏
